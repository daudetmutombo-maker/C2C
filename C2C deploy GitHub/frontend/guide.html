Manuel d'Utilisation et d'Administration C2C

Bienvenue dans la version enrichie du guide Pilotage Stratégique C2C. Ce manuel couvre les fonctionnalités de personnalisation, de sécurité et de gestion stratégique implémentées pour la cellule stratégique.

NOTE DE SÉCURITÉ

L'application utilise désormais une authentification par JWT (jetons signés) avec mots de passe hachés (bcrypt). Toutes les routes API sont protégées par un middleware qui vérifie le token à chaque requête.

MANUEL D'ADMINISTRATION (POUR LE SERVICE IT)

1) Déploiement et Configuration

  • Base de Données : SQLite locale (backend/c2s_database.db). Pour PostgreSQL, modifier DATABASE_URL dans backend/.env.
  • Configuration : Un fichier backend/.env centralise la configuration (clé secrète JWT, expiration du token, origines CORS, URL de base de données).
  • Structure du Backend : Le code est organisé en modules : config.py (configuration), auth.py (JWT, hash), models.py (ORM), schemas.py (validation), main.py (routes).

2) Démarrage du Serveur

  1. cd backend
  2. pip install -r requirements.txt
  3. python -m uvicorn main:app --reload

Le serveur démarre sur http://127.0.0.1:8000. Les utilisateurs par défaut sont créés automatiquement au premier lancement.

4) Sécurité

  • Authentification : Toutes les routes /api/* (sauf /api/login) sont protégées par un middleware.
  • Mots de passe : Hachés avec bcrypt avant stockage.
  • Tokens JWT : Signés avec une clé secrète (configurable dans .env), expiration par défaut de 8 heures.
  • CORS : Configurable via CORS_ORIGINS dans .env.
  • Environnement : Le fichier .env contient les secrets et n'est pas versionné (présent dans .gitignore).

IMPORTANT SÉCURITÉ

Changez la clé secrète SECRET_KEY dans backend/.env avant tout déploiement en production. En environnement réel, passez sur PostgreSQL et activez HTTPS.

MANUEL UTILISATEUR (POUR LE CABINET)

A. Connexion à l'Application

  1. Ouvrez votre navigateur et allez sur http://127.0.0.1:8080 (ou l'adresse du serveur).
  2. Vous êtes redirigé vers la page de connexion.
  3. Saisissez votre nom d'utilisateur et mot de passe fournis par le service IT.
  4. Cliquez sur S'identifier. Si les identifiants sont corrects, vous accédez au tableau de bord.

B. Personnalisation Visuelle (Thèmes)

  • Galerie de Thèmes : Cliquez sur "Paramètres" pour choisir parmi des ambiances prédéfinies (Business Blue, Drapeau RDC, Deep Night, Émeraude).
  • Réglages Avancés : Vous pouvez affiner chaque couleur manuellement via l'onglet "Réglages avancés".
  • Persistance : Vos choix de couleurs sont mémorisés même après fermeture du navigateur.
  • Export/Import : Sauvegardez et restaurez votre configuration visuelle via les boutons dédiés.

C. Rapports Stratégiques

  • Édition Libre : Les rapports générés automatiquement sont modifiables. Vous pouvez cliquer dans la zone de texte pour ajouter vos propres analyses avant de télécharger ou d'imprimer.
  • Types de Rapports : Quotidien, Mensuel, Semestriel, Annuel.
  • Impression : Le bouton "Imprimer" formate le document pour une présentation claire en réunion.

D. Gestion des Données (Export/Import JSON)

  • Export : Télécharge un fichier .json contenant toutes les décisions pour archivage.
  • Import : Restaure des décisions depuis un fichier JSON précédemment exporté.
  • Sauvegarde automatique : Le système sauvegarde les données en local toutes les 2 minutes.
  • Mode Hors-Ligne : En cas de perte de connexion, les données restent accessibles depuis le cache. Les modifications sont synchronisées automatiquement au retour de la connexion.

E. Aide Interactive (Assistant C2C)

Situé dans le menu latéral, l'assistant vous guide pas à pas :

  1. Cliquez sur l'assistant pour ouvrir le menu des tutoriels.
  2. Sélectionnez une action (ex: "Nouvelle Décision", "Corriger une Erreur").
  3. Observez la souris virtuelle se déplacer et cliquer pour vous montrer la marche à suivre.

F. Messagerie Réseau

  • Accès : Depuis le menu latéral, cliquez sur "Messagerie".
  • Envoi : Saisissez votre nom et votre message, puis cliquez sur "Envoyer".
  • Réception : Les messages se rafraîchissent automatiquement toutes les 10 secondes.
